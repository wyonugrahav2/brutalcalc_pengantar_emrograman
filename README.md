BRUTALCALC

Soviet Computational Machine  
Modern brutalist calculator built with Flask.

Features

- Arithmetic operations
  - Add
  - Subtract
  - Multiply
  - Divide
  - Modulus
  - Floor Division
  - Power
  - Square Root

- Logic gates
  - AND
  - OR
  - NOT
  - XOR
  - NAND
  - NOR

- Transformations
  - Base conversion
  - Temperature conversion
  - Currency conversion
  - Factorial
  - Fibonacci

- Persistent history
- Dark / Light mode
- Brutalist Soviet UI
- JSON storage system

Tech Stack

- Python Flask
- HTML Jinja Templates
- Vanilla JavaScript
- CSS Brutalist Design
- JSON Local Persistence

Installation
Clone repository:
bash
git clone https://github.com/USERNAME/brutalcalc_pengantar_emrograman.git
cd brutalcalc_pengantar_emrograman


Install dependencies:
bash
pip install -r requirements.txt


Run app:
bash
python app.py

Open:
txt
http://127.0.0.1:5000
